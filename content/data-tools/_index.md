DbGate offers several advanced data tools suitable for specific scenarios:

- [Data archives](archives) - table data stored in folders (on NDJSON format) on your local computer. You could browse and edit this data in the similar way as browsing tables in database
- [Data deployer](data-deployer) - visually compare and synchronize data between data archive and real database
- [Data replicator](data-replicator) - replicate your data from data archive to database, suitable for run from DbGate app or from command line
